What Is Voxa?

Voxa is a Programming Language being made by BluePoint. The mission of this programming language is to make a simple and easy to learn programming language that has a lot of power and functionality. Voxa is being built using Python.

Why Voxa?

Voxa was made out of the thought of the complexity of some languages and to provide an easy and straightforward experience for aspiring developers around the world.
